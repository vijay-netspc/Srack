The program must accept an integer matrix of size NxN as the input. The integers in
the first row of the matrix are always unique and their values range from 1 to N. For
each integer X in the first row of the matrix, the program must print the X'" row as a
column as the output.

Boundary Condition(s):
2<=N<=50

Input Format:
The first line contains N.
The next N lines, each contains N integers separated by a space.

Output Format:
The first N lines, each contains N integers separated by a space.

Example Input/Output 1:
Input:
3
3 1 2 
8 9 10
4 2 6

Output:
4 3 8
2 1 9
6 2 10
